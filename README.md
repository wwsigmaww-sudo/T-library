# T-Библиотека

Это простая программа для управления своими книгами.  
Работает в консоли (чёрное окно с текстом).

## Что умеет программа?

- **Добавлять книгу** - вводишь название, автора, жанр, год и описание.
- **Смотреть список книг** - можно отсортировать по названию, автору или году, а также отфильтровать по жанру или по тому, прочитана книга или нет.
- **Добавлять в избранное** - помечаешь книгу звёздочкой, чтобы потом быстро её найти.
- **Убирать из избранного** - если книга больше не любимая.
- **Отмечать прочитанное** - статус меняется с «не прочитана» на «прочитана» и обратно.
- **Смотреть избранное** - показывает только те книги, которые ты добавил в избранное.
- **Удалять книгу** - если она больше не нужна в библиотеке.
- **Искать книгу** - по слову из названия, автора или описания.
